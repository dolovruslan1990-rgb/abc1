# Movie Library

**Автор:** Псануков Амир

## Описание

Movie Library — это графическое приложение для управления личной коллекцией фильмов. Программа позволяет:
* Добавлять фильмы с указанием названия, жанра, года выпуска и рейтинга.
* Просматривать все фильмы в виде таблицы.
* Фильтровать коллекцию по жанру и году выпуска.
* Сохранять данные в JSON‑файл для постоянного хранения.

## Функциональность

* **Добавление фильмов:** форма с валидацией данных (проверка года и рейтинга).
* **Отображение:** таблица с возможностью просмотра всей коллекции.
* **Фильтрация:** поиск по жанру (выпадающий список) и году (точное совпадение).
* **Сохранение:** данные сохраняются в `movies.json`, что позволяет сохранять коллекцию между запусками.

1. Убедитесь, что у вас установлен Python 3.x.
2. Установите зависимости (если нужны). В данном случае дополнительных зависимостей нет.
3. Скачайте или клонируйте репозиторий:
   ```bash
   git clone [URL-репозитория]
4. Перейдите в папку проекта:
   ```bash
   cd movie_library
5. Запустите приложение:
   ```bash
   python main.py

## Примеры использования
### **Тест 1: Добавление фильма**
1. Заполните поля:
   * Название: «Интерстеллар»
   * Жанр: «Фантастика»
   * Год выпуска: 2014
   * Рейтинг: 8.6
2. Нажмите «Добавить фильм».
3. Проверьте, что фильм появился в таблице.

### **Тест 2: Фильтрация по жанру**
1. В фильтре выберите жанр «Фантастика».
2. Нажмите «Применить фильтр».
3. Убедитесь, что отображаются только фильмы этого жанра.

### **Тест 3: Фильтрация по году**
1. Введите в поле «Год» значение 2014.
2. Нажмите «Применить фильтр».
3. Убедитесь, что отображаются фильмы 2014 года.

### **Тест 4: Сброс фильтров**
1. Нажмите «Сбросить фильтр».
2. Убедитесь, что отображается вся коллекция.
